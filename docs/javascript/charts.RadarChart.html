Class: RadarChart

charts.RadarChart(props, margin, chartWidth, chartHeight, chartCenter, chartRadius, axisScale)

Render a radarchart (spiderchart). Exported with curry to ensure the props from CharContainer are well checked

Constructor

new RadarChart(props, margin, chartWidth, chartHeight, chartCenter, chartRadius, axisScale)

Parameters:
Name Type Description
props object
Properties
Name Type Description
title string the title of the chart
axis array the details of the Axis data (key and name)
scales array the value of the ticks to build the grid
dataset array the processed data to make the chart. Array of object, with a key "x" containing the axis index and a key "y" containing a y value.
containerWidth number the width of the container provided by ChartContainer
containerHeight number the height of the container provided by ChartContainer
margin object the margin top, bottom, left, right for the chart inside its container
chartWidth number the width of the chart (container - margins)
chartHeight number the height of the chart (container - margins)
chartCenter object the x and y coordinates of the chart center
chartRadius number the radius of the chart (min between chartWidth/2 and chartHeight/2)
axisScale function the scale function to convert data into position/length, updated in this.updateChart()
