Class: BarChart

charts.BarChart(props, margin, chartWidth, chartHeight, yAxisSerieIndex, headerMargin)

Render a barchart. Exported with curry to ensure the props from CharContainer are well checked

Constructor

new BarChart(props, margin, chartWidth, chartHeight, yAxisSerieIndex, headerMargin)

Parameters:
Name Type Description
props object
Properties
Name Type Description
title string the title of the chart
xAxis object the details of the x Axis data (name, unit)
series array is a table of object, each one contains the details of the data serie (name, unit, color, starting from zero or not, y axis displayed or not). Make sure the sequence of serie is the same between this and series and dataset
dataset array the processed data to make the barchart. Array of object, with a key "x" containing x value and a key "y" containing a table with a y value for each serie.
containerWidth number the width of the container provided by ChartContainer
containerHeight number the height of the container provided by ChartContainer
margin object the margin top, bottom, left, right for the chart inside its container
chartWidth number the width of the chart (container - margins)
chartHeight number the height of the chart (container - margins)
yAxisSerieIndex number the index of the serie to use to build the y axis
headerMargin number the margin.top for both title and legend
