Class: LineChart

charts.LineChart(props, margin, chartWidth, chartHeight, xAxisScale, yAxisScale)

Render a linechart. Exported with curry to ensure the props from CharContainer are well checked

Constructor

new LineChart(props, margin, chartWidth, chartHeight, xAxisScale, yAxisScale)

Name Type Description
props object
Properties
Name Type Description
title string the title of the chart
xAxis object the details of the x Axis data (name, unit)
serie object the details of the data serie (name, unit, color)
dataset array the processed data to make the linechart. Array of object, with a key "x" containing x value and a key "y" containing a y value.
xAxisLabel function a function to convert x data into readable label for the axis
containerWidth number the width of the container provided by ChartContainer
containerHeight number the height of the container provided by ChartContainer
margin object the margin top, bottom, left, right for the chart inside its container
chartWidth number the width of the chart (container - margins)
chartHeight number the height of the chart (container - margins)
xAxisScale function the x scale function to convert data into position/length, updated in this.updateChart()
yAxisScale function the y scale function to convert data into position/length, updated in this.updateChart()
